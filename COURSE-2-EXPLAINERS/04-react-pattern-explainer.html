What is ReAct?

ReAct (Reasoning + Acting) is a pattern where AI agents combine reasoning (thinking through problems step-by-step) with acting (calling tools/functions) to solve complex tasks. The agent alternates between:

  • Thought: Reasoning about the current situation and planning next steps
  • Action: Executing a tool or function to gather information
  • Observation: Receiving and processing the results

This cycle repeats until the agent has enough information to provide a final answer.

Why ReAct Matters

Traditional AI approaches either generate one-shot responses or act blindly. ReAct agents are:

  • Adaptive: Can adjust plans based on new information
  • Transparent: Expose their reasoning process
  • Reliable: Verify information step-by-step
  • Efficient: Make fewer irrelevant or incorrect actions

Step 3: The ReAct Agent Loop

class ReActAgent {
  constructor(apiKey, tools, functions) {
    this.apiKey = apiKey;
    this.tools = tools;
    this.functions = functions;
    this.maxIterations = 10;
  }

  async runReActLoop(task) {
    let iteration = 0;
    let finished = false;
    let scratchpad = ""; // Agent's thinking space
    let finalAnswer = null;

    const messages = [
      {
        role: "system",
        content: `You are a ReAct agent. Follow this format strictly:
Thought: reason about what to do next
Action: choose a tool to call (name and args in JSON)
Observation: wait for tool result

Or:
Thought: reason about final answer
Final Answer: provide your answer

Continue looping until you have enough information.`
      },
      { role: "user", content: task }
    ];

    while (!finished && iteration < this.maxIterations) {
      iteration++;

      console.log(`\\n--- Iteration ${iteration} ---`);

      const response = await this.callAI(messages, scratchpad);
      messages.push({ role: "assistant", content: response });

      console.log(`AI Response: ${response}`);

      // Parse the ReAct format
      const thoughtMatch = response.match(RegExp('Thought: (.*?(?=Action:|Final Answer:|\\n|$))', 's'));
      const actionMatch = response.match(RegExp('Action: (.*?(?=Observation:|Final Answer:|\\n|$))', 's'));
      const finalMatch = response.match(RegExp('Final Answer: (.*)', 's'));

      if (finalMatch) {
        finalAnswer = finalMatch[1].trim();
        finished = true;
        console.log(`Final Answer: ${finalAnswer}`);
        continue;
      }

      if (!actionMatch) {
        console.log("No action found, agent might be stuck");
        continue;
      }

      // Extract tool call from Action
      const actionText = actionMatch[1].trim();
      let toolCall;
      try {
        toolCall = JSON.parse(actionText);
      } catch (e) {
        console.log("Failed to parse action", actionText);
        continue;
      }

      const functionName = toolCall.name;
      const functionArgs = toolCall.arguments;

      console.log(`Executing: ${functionName}(${JSON.stringify(functionArgs)})`);

      // Execute the function
      const func = this.functions[functionName];
      if (!func) {
        console.log(`Unknown function: ${functionName}`);
        continue;
      }

      const result = await func(...Object.values(functionArgs));

      console.log(`Result: ${result}`);

      // Add observation to scratchpad and messages
      const observation = `Observation: ${result}`;
      scratchpad += response + observation + "\n";
      messages.push({ role: "user", content: observation });
    }

    return finalAnswer;
  }

  async callAI(messages, scratchpad) {
    const allMessages = [
      ...messages,
      { role: "user", content: scratchpad || "Start reasoning..." }
    ];

    const response = await fetch("https://api.openai.com/v1/chat/completions", {
      method: "POST",
      headers: {
        "Content-Type": "application/json",
        "Authorization": `Bearer ${this.apiKey}`
      },
      body: JSON.stringify({
        model: "gpt-4",
        messages: allMessages,
        tools: this.tools,
        tool_choice: "auto",
        temperature: 0.1 // Low temperature for consistent reasoning
      })
    });

    const data = await response.json();
    return data.choices[0].message.content;
  }
}

Key Components of ReAct Implementation

1 Structured Prompting

Guide the AI with specific instructions on how to format thoughts, actions, and observations.

"Follow this format strictly:
Thought: reason about what to do next
Action: choose a tool (name and args in JSON)

2 Scratchpad Memory

Maintain conversation history including previous thoughts, actions, and observations.

3 Loop Control

Implement maximum iteration limits and stopping conditions to prevent infinite loops.

4 Result Parsing

Use regex or structured parsing to extract thoughts, actions, and final answers from AI responses.

Best Practices

‚úÖ Provide Clear Instructions

Use detailed system prompts that explain the ReAct format and when to stop.

‚úÖ Implement Safety Limits

Always set maximum iterations and timeout limits to prevent runaway agents.

‚úÖ Handle Errors Gracefully

If a tool fails, the agent can try alternative approaches in subsequent iterations.

‚úÖ Use Low Temperature

Set temperature 0.1-0.3 for consistent reasoning patterns in the loop.

‚úÖ Provide Good Tool Descriptions

Well-described tools help agents make better decisions about which actions to take.
